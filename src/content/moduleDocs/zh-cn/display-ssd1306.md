# SSD1306 OLED 显示模块

SSD1306 是一款常用的 OLED 显示驱动芯片，广泛应用于各类小型显示模块。它支持单色图形显示，具有超低功耗和高对比度的特点。

## 主要特性

- **超低功耗**：正常工作电流仅为几微安，非常适合电池供电设备
- **高对比度**：自发光技术，对比度远超传统 LCD
- **宽视角**：接近 180 度的视角范围，几乎无视觉死角
- **快速响应**：微秒级响应时间，无拖影
- **多种分辨率**：支持 128x64、128x32、96x16 等多种分辨率
- **灵活接口**：支持 I2C、SPI、6800/8080 并行接口

## 应用场景

- 智能穿戴设备（手表、手环）
- 便携式仪器面板
- 电子标签
- 物联网设备状态显示
- DIY 项目

## 技术规格

| 参数 | 值 |
|------|-----|
| 驱动芯片 | SSD1306 |
| 显示颜色 | 单色（蓝色/白/黄） |
| 工作电压 | 3.3V - 5V |
| 工作温度 | -30°C ~ +70°C |
| 响应时间 | < 10μs |
| 刷新率 | > 60fps |

## 使用建议

1. **上电顺序**：确保 VCC 稳定后再连接 I2C/SPI 信号
2. **初始化**：每次上电后需要进行完整的初始化序列
3. **显示缓冲区**：SSD1306 需要外部 RAM 作为显示缓冲区
4. **亮度调节**：通过预充电电阻设置可实现亮度调节

## 常见问题

**Q: 为什么屏幕是全黑的？**
A: 检查初始化序列是否完整，特别是显示开启指令。

**Q: 为什么显示的字符有残影？**
A: 适当调整预充电和设置段引脚重映射参数。

**Q: I2C 地址是多少？**
A: 默认地址为 0x3C（SA0 接地）或 0x3D（SA0 接高）。
