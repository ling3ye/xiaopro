みんながローカルで AI を実行することに興味を持っているようなので、今日は面白いプロジェクトを紹介します——**ACE-Step 1.5**。

簡単に言うと、自分の Mac で音楽を生成できます。完全無料、インターネット接続なしでも実行可能で、このプロジェクトでは Apple チップのパフォーマンス最適化もしっかり行われています。

難しく考える必要はありません。コマンドを数行入力するだけです。私の手順に従えば、数分で完了します！

## 準備作業

ターミナル（Terminal）を開いて、始めましょう。

### 1. 「巣」を作る

まず、プロジェクトを保存する場所を決めます。

```bash
cd Projects
cd Python
```

### 2. コードをクローン & 依存関係をインストール

GitHub からプロジェクトを直接クローンして、`uv` を使って依存関係を素早くインストールします（`uv` をまだ使ったことがない場合、ぜひインストールをおすすめします。Python 環境管理の最強ツールです）。

```bash
git clone https://github.com/ACE-Step/ACE-Step-1.5.git
cd ACE-Step-1.5
uv sync
```

*（ここで少し待って、プログレスバーが完了するのを待ちます。）*

### 3. Mac の「馬力」を確認

インストール後、すぐに実行せず、Apple チップ（MPS アクセラレーション）が正しく認識されているか確認しましょう。

```bash
uv run python -c "import torch; print(f'MPS check: {torch.backends.mps.is_available()}')"
```

ターミナルに **`MPS check: True`** と返ってくれば、すべて正常で、GPU の準備完了です！

## 起動と実行

### 4. サービスを起動

特に説明することなく、直接実行します：

```bash
uv run acestep
```

### 5. ブラウザで開く

ターミナルで実行されたら、ブラウザを開いて：`127.0.0.1:7860` を入力します。

ACE-Step の操作画面が表示されます。

**ここで注意すべき重要なポイント：**

- **設定を正しく選択**：画面でモデルの VRAM サイズを選択します。自分の Mac のユニファイドメモリの容量に応じて選んでください。デモでは **16-20GB** を選択しています。
- **初期化**：**「Initialize Service」** をクリックします。
  - ⚠️ 注意：最初の実行時にモデルが自動的にダウンロードされます。ファイルサイズが大きいため、ここで一時的に止まることがありますが、正常な動作です！コーヒーを飲みながら、セットアップが完了するのを待ちましょう。

### 6. 最初の AI 音楽を生成

環境が整ったら、操作は非常にシンプルです：

1. **「Simple」** モードに切り替えます。
2. プロンプト（Prompt）を入力します。何を書けばいいかわからない場合は `easy example` と入力してください。
3. **「Create Sample」** をクリックします。下に複雑なパラメータ（Custom の内容）が自動的に入力されますが、気にする必要はありません。
4. 最後の **「Generate Music」** をクリックします。

完了！少し待って、プログレスバーが終了すれば、ローカル GPU で「計算」された音楽を聞くことができます 🎵
