## 实验简介

BME280 是 Bosch 推出的一款高精度环境传感器，可以同时测量温度、湿度和气压。本实验将使用 Pico 通过 I2C 接口读取其数据。

## 硬件连接

| BME280 | Raspberry Pi Pico |
|--------|-------------------|
| VCC | 3.3V |
| GND | GND |
| SCL | GP5 |
| SDA | GP4 |

## 代码示例 (MicroPython)

```python
import machine
import bme280

i2c = machine.I2C(0, sda=machine.Pin(4), scl=machine.Pin(5))
bme = bme280.BME280(i2c=i2c)

while True:
    temperature, pressure, humidity = bme.read_compensated_data()
    temp_c = temperature / 100
    pres_hpa = pressure / 256
    hum = humidity / 1024

    print(f"温度: {temp_c:.1f}°C")
    print(f"气压: {pres_hpa:.0f}hPa")
    print(f"湿度: {hum:.1f}%")
    print("-" * 20)

    import time
    time.sleep(2)
```

## 安装库

使用 Thonny IDE，将 bme280.py 文件复制到 Pico 的文件系统中。
